screenshot-as-a-service
=======================

> Website screenshot service powered by [Node.js][nodejs] and and [Google Chrome Pupeeter][pupeeter].


Requirements
------------

- [Node.js][nodejs] with [NPM][npm] v5+


Installation
------------

```sh
npm install
```
